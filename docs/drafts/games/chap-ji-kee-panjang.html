Chap Ji Ki Panjang

Game info

Players: any
Country: Singapore

Categories

    Posted

    August 12, 2018

    Chap Ji Ki Panjang (十二支現場?) is a betting game from Singapore that is played with Four Colour Cards. The gameplay is very similar to that of the Japanese Tehonbiki.

    Equipment

    Each player needs 12 cards, one of each type from a Four Colour Cards deck.
