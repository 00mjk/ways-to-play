Analysis

Solving Teeko was suggested as Item 90 in the “HAKMEM” memo (Beeler, Gosper, and Schroeppel 1972). Guy Steele proved that the basic game is a draw if played perfectly, and that the advanced game is a first-player win (Steele 1998).
